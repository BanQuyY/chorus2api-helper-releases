# Chorus2api-Helper 

## 部署步骤

### Mac 环境运行
```bash
./chorus2api-helper
```

![img.png](docs/img.png)

### 内网穿透 [可选]
如需外网访问，可配置内网穿透服务。

## [Chorus2api](https://2api-docs.pages.dev/page/chorus2api/quick-deploy.html) 配置

### 添加环境变量
```
CHEAT_URL=http://192.168.124.20:11022
```
> 注意：如使用内网穿透，需填写对应的域名地址
